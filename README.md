# Sepiru AI 🚀

SaaS-based AI-powered data intelligence platform for CSV and Excel datasets.

🔗 Live Demo: https://sepiru-ai.streamlit.app

## Features
- 📊 Data Profiling
- 🤖 AI Chat with Data (LLM-powered)
- 📈 Forecasting & ML Modeling
- 📉 Visualization & Insights

## Tech Stack
- Streamlit 
- Groq 
- Python, Pandas
